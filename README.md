# Blog Website Template With Vue.js

Blog Website Template built with Vue.js

## Objectives

Why Build This?

1. To Learn Vue.js.
2. Nothing Else.

## Project setup

```
$ npm install

// Compiles and hot-reloads for development
$ npm run serve

// Compiles and minifies for production
$ npm run build

// Lints and fixes files
$ npm run lint

```
