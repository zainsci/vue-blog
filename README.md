# Personal Blog With VUE.js

Personal Blog website build with VUE.js

## Objectives

1. Learn VUE.js

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
